# Simple YouTube UI

This project is a simple YouTube user interface clone made with HTML. It mimics the basic layout and structure of YouTube's homepage.


## Features

- Responsive design
- Header with logo and search bar
- Sidebar with navigation icons
- Main content area displaying video thumbnails
- Footer with basic information

## Installation

To get started with this project, you can clone the repository to your local machine using Git:
